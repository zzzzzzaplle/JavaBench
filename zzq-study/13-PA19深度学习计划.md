# PA19 深度学习计划

## 学习目标

逐文件、逐函数、逐行代码理解 PA19 管道游戏项目。

---

## 学习路径（按依赖关系从底到顶）

```
第1层：工具层 - 无依赖
┌─────────────────────────────────────────┐
│  util/Direction.java      - 方向枚举      │
│  util/Coordinate.java     - 坐标类        │
│  util/PipePatterns.java   - 管道字符常量  │
│  util/StringUtils.java    - 字符串工具    │
└─────────────────────────────────────────┘
                    ↓
第2层：核心抽象层
┌─────────────────────────────────────────┐
│  game/MapElement.java     - 地图元素接口  │
│  game/map/cells/Cell.java - 格子抽象类    │
└─────────────────────────────────────────┘
                    ↓
第3层：具体实现层
┌─────────────────────────────────────────┐
│  game/map/cells/Wall.java          - 墙壁│
│  game/map/cells/FillableCell.java  - 可填充格│
│  game/map/cells/TerminationCell.java - 起点/终点│
│  game/pipes/Pipe.java              - 管道  │
└─────────────────────────────────────────┘
                    ↓
第4层：游戏组件层
┌─────────────────────────────────────────┐
│  game/PipeQueue.java  - 管道队列          │
│  game/CellStack.java  - 撤销栈            │
│  game/DelayBar.java   - 延迟计时器        │
│  game/map/Map.java    - 地图（核心！）     │
└─────────────────────────────────────────┘
                    ↓
第5层：游戏控制层
┌─────────────────────────────────────────┐
│  game/Game.java       - 游戏主控制器      │
│  io/Deserializer.java - 文件解析器        │
│  Main.java            - 程序入口          │
└─────────────────────────────────────────┘
```
