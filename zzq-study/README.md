### 推荐立即行动：

1. **环境搭建**（Day 1-2）
   ```bash
   git clone https://github.com/java-bench/JavaBench.git
   pip install -r requirements.txt
   ```

2. **快速测试**（Day 3-4）
   ```bash
   python inference.py --model-path gpt-3.5-turbo \
       --data datasets/selective-context/data-PA19.jsonl \
       --output output/test/samples.jsonl
   ```

3. **评估分析**（Day 5-7）
   ```bash
   python evaluation.py class-wise \
       --output output/test/results.json \
       output/test/samples.jsonl
   ```
