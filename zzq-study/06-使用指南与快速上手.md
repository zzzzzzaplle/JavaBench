### 3.3 推理参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `--model-path` | str | 必需 | 模型路径或名称 |
| `--data` | str | 必需 | 输入数据集路径 |
| `--output` | str | 必需 | 输出文件路径 |
| `--mode` | str | holistic | 合成策略 |
| `--num-sample` | int | 10 | 每个任务的样本数 |
| `--temperature` | float | 0.2 | 采样温度 |
| `--repetition_penalty` | float | 1.0 | 重复惩罚 |
| `--max-new-tokens` | int | 4096 | 最大生成 token 数 |

### 3.4 合成策略选择

```bash
# 整体合成（推荐）
--mode holistic

# 独立合成（每个方法独立生成）
--mode independent

# 增量合成（顺序）
--mode incremental --incremental-mode seq

# 增量合成（逆序）
--mode incremental --incremental-mode rev

# 增量合成（随机）
--mode incremental --incremental-mode rand
```

### 4.3 批量评估脚本

```bash
#!/bin/bash
# evaluate_all.sh

PROJECTS=("PA19" "PA20" "PA21" "PA22")
MODEL="gpt-3.5-turbo"

for PROJECT in "${PROJECTS[@]}"; do
    echo "========================================="
    echo "Processing $PROJECT..."
    echo "========================================="
    
    # 创建输出目录
    mkdir -p output/result-$PROJECT/$MODEL
    
    # 1. 推理
    echo "Running inference..."
    python inference.py \
        --model-path $MODEL \
        --num-sample 5 \
        --output output/result-$PROJECT/$MODEL/samples.jsonl \
        --data datasets/selective-context/data-$PROJECT.jsonl \
        --mode holistic
    
    # 2. 类级别评估
    echo "Evaluating class-wise..."
    python evaluation.py class-wise \
        --output output/result-$PROJECT/$MODEL/single_class.json \
        output/result-$PROJECT/$MODEL/samples.jsonl
    
    # 3. 测试级别评估
    echo "Evaluating test-wise..."
    python evaluation.py test-wise \
        --output output/result-$PROJECT/$MODEL/result-full.json \
        --test datasets/testcase/test-$PROJECT.jsonl \
        output/result-$PROJECT/$MODEL/samples.jsonl
    
    echo "$PROJECT completed!"
    echo ""
done

echo "All projects evaluated successfully!"
```

## 五、结果分析

### 5.1 计算指标

```python
import json
from scipy.special import comb

def calculate_metrics(results_file):
    """计算所有指标"""
    with open(results_file, 'r') as f:
        results = json.load(f)
    
    total = len(results)
    
    # Completion@k
    can_replace = sum(1 for r in results if r['can_replace'])
    completion_rate = can_replace / total
    
    # Compilation@k
    compile_success = sum(1 for r in results if r['compile_errors'] == 0)
    compilation_rate = compile_success / total
    
    # Pass@k (k=1)
    test_pass = sum(1 for r in results if r['test_result'][0] == r['test_result'][1])
    pass_rate = test_pass / total
    
    return {
        'total': total,
        'completion_rate': f"{completion_rate:.2%}",
        'compilation_rate': f"{compilation_rate:.2%}",
        'pass_rate': f"{pass_rate:.2%}"
    }

# 使用示例
metrics = calculate_metrics('output/result-PA19/gpt-3.5-turbo/single_class.json')
print(json.dumps(metrics, indent=2))
```

**输出示例**：
```json

  "total": 17,
  "completion_rate": "88.24%",
  "compilation_rate": "64.71%",

}
```

### 5.2 Pass@k 计算

```python
def pass_at_k(n, c, k):
    """
    计算 Pass@k
    
    Args:
        n: 总样本数
        c: 通过的样本数
        k: Pass@k 的 k 值
    """
    if n - c < k:
        return 1.0
    return 1.0 - comb(n - c, k) / comb(n, k)

def calculate_pass_at_k(results, k_values=[1, 5, 10]):
    """为每个任务计算 Pass@k"""
    # 按 task_id 分组
    from collections import defaultdict
    task_results = defaultdict(list)
    
    for result in results:
        task_results[result['task_id']].append(result)
    
    # 计算每个任务的 Pass@k
    metrics = {}
    for task_id, samples in task_results.items():
        n = len(samples)
        c = sum(1 for s in samples if s['test_result'][0] == s['test_result'][1])
        
        metrics[task_id] = {
            f'pass@{k}': pass_at_k(n, c, k) for k in k_values
        }
    
    return metrics

# 使用示例
with open('output/result-PA19/gpt-3.5-turbo/single_class.json', 'r') as f:
    results = json.load(f)

metrics = calculate_pass_at_k(results)
print(json.dumps(metrics, indent=2))
```

### 5.3 可视化结果

```python
import json
import matplotlib.pyplot as plt
import seaborn as sns

def visualize_results(results_file, output_file='results.png'):
    """可视化评估结果"""
    with open(results_file, 'r') as f:
        results = json.load(f)
    
    # 提取数据
    tasks = [r['task_id'].split('/')[1] for r in results]
    compile_success = [1 if r['compile_errors'] == 0 else 0 for r in results]
    test_pass_rate = [
        r['test_result'][0] / r['test_result'][1] if r['test_result'][1] > 0 else 0
        for r in results
    ]
    
    # 创建图表
    fig, axes = plt.subplots(2, 1, figsize=(14, 10))
    
    # 编译成功率
    colors = ['green' if x == 1 else 'red' for x in compile_success]
    axes[0].bar(range(len(tasks)), compile_success, color=colors)
    axes[0].set_title('Compilation Success by Task', fontsize=14)
    axes[0].set_ylabel('Success (0/1)', fontsize=12)
    axes[0].set_xticks(range(len(tasks)))
    axes[0].set_xticklabels(tasks, rotation=45, ha='right')
    axes[0].grid(axis='y', alpha=0.3)
    
    # 测试通过率
    colors = plt.cm.RdYlGn(test_pass_rate)
    axes[1].bar(range(len(tasks)), test_pass_rate, color=colors)
    axes[1].set_title('Test Pass Rate by Task', fontsize=14)
    axes[1].set_ylabel('Pass Rate', fontsize=12)
    axes[1].set_xlabel('Task', fontsize=12)
    axes[1].set_xticks(range(len(tasks)))
    axes[1].set_xticklabels(tasks, rotation=45, ha='right')
    axes[1].set_ylim([0, 1])
    axes[1].grid(axis='y', alpha=0.3)
    
    plt.tight_layout()
    plt.savefig(output_file, dpi=300)
    print(f"Results saved to {output_file}")

# 使用示例
visualize_results(
    'output/result-PA19/gpt-3.5-turbo/single_class.json',
    'pa19_results.png'
)
```

```python
{code_context}

Complete this Java class:

{code}

3. **Maintain OOP principles**:
```

## 七、常见问题与解决方案

### 7.1 编译错误

**问题**：生成的代码编译失败

**解决方案**：
```bash
# 查看详细编译错误
cd projects/PA19-Solution
./gradlew compileJava --info

# 检查 Java 版本
java -version
```

### 7.2 内存不足

**问题**：评估时内存不足

**解决方案**：
```bash
# 增加 JVM 堆内存
export GRADLE_OPTS="-Xmx4g"

# 或在 gradle.properties 中设置
echo "org.gradle.jvmargs=-Xmx4g" >> gradle.properties
```

### 7.3 测试超时

**问题**：测试运行超时

**解决方案**：
```bash
# 增加超时时间（在 build.gradle 中）
test {
    timeout = Duration.ofMinutes(5)
}
```

### 7.4 输出过长

**问题**：LLM 生成的代码被截断

**解决方案**：
```bash
# 增加最大 token 数
python inference.py \
    --max-new-tokens 8192 \
    ...
```
